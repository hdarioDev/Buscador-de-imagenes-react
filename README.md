# The searched images with React

App react js, html, css responsive, paginator, mode dark, viewport, etc

# Api Reference

https://www.pixabay.com/

# Demo 

https://resplendent-bunny-9a939b.netlify.app

## Install

```
npm install
```

## Run for development

```
npm run start
```

## Build to production

```
npm run build
```
